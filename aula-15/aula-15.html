Operacoes Logicas

Maior que (>)

Dados os numeros 'a' e 'b', a operacao 'a > b' ira' retornar 'True' se e somente se a for maior que b. caso contrario a operacao ira retornar falso.

Exemplo:

let a = 5;
let b = 1
console.log(a > b) //True
console.log(b > a) //False

Menor que (<)

Dados os numeros 'a' e 'b', a operacao 'a < b' ira' retornar 'True' se e somente se 'a' for menor que 'b'. caso contrario a operacao ira retornar falso.

Exemplo:

let a = 5;
let b = 10
console.log(a < b) //True
console.log(b < a) //False

Maior ou igual a (>=)

Dados os numeros 'a' e 'b', a operacao 'a >= b' ira' retornar 'True' se e somente se 'a' for maior ou igual a 'b'. caso contrario a operacao ira retornar falso.

Exemplo:

let a = 5;
let b = 1
let c = 5
console.log(a >= b) //True
console.log(a >= c) //True

Menor ou igual a (<=)

Dados os numeros 'a' e 'b', a operacao 'a <= b' ira' retornar 'True' se e somente se 'a' for Menor ou igual a 'b'. caso contrario a operacao ira retornar falso.

Exemplo:

let a = 5;
let b = 1
let c = 5
console.log(a <= b) //False
console.log(a <= c) //True

Igual a (== e ===)

Dados os numeros 'a' e 'b', a operacao 'a == b' ou 'a === b' ira retornar verdade se e somente se 'a' for igual a 'b';

Diferenca entre '=', '==' e '==='

Usamos '=' quando queremos taribuir um valor a uma variavel;

Exemplo:

let x = 5;
console.log(x = 10); //10

Usamos '===' para comparar dois valores do mesmo tipo.

Exemplo:

let x = 10;
console.log(x === 10); //true
Neste caso, a variavel x e o numero 10 sao do mesmo tipo (number)

Usamos '==' para comparar dois valores independentemente do tipo

Exemplo:

let x = 10;
console.log(x == "10"); //true
console.log(x === "10"); //false
console.log(x == "20"); //false
Neste caso, a variavel x e o numero 10 sao do mesmo tipo (number)

Obs: E' sempre aconselhado usar === para evitar erros de comparacao.

And (&&)

Dados os vaores 'a' e 'b', a operacao 'a && b' retorna verdade se e somente se os dois valores forem verdadeiros. Se um deles for falso a operacao ira retornar falso.

Exemplo:

let a = true;
let b = true
let c = false
console.log(a && b) //true
console.log(a && c) //false

OR (||)

Dados os vaores 'a' e 'b', a operacao 'a || b' retorna verdade se pelo menos um dos dois valores for verdadeiro. Caso contrario a operacao ira retornar falso.

Exemplo:

let a = true;
let b = true
let c = false
console.log(a || b) //true
console.log(a || c) //true
console.log(c || false) //false

NOT (!)

Dado um valor a, a operacao '!a' retorna o oposto do valor inicial de a.

Exemplo:

let a = true;
console.log(!a) //false

Estruturas de decisao

As estruturas de decisao nos permitem executar uma determinada porcao de codigo dependendo de uma dada condicao

if

O if nos permite executar um codigo se e somente se uma data condicao for satisfeita

Exemplo:

let idade = 21;
if(idade >= 18) {
console.log("Ja pode tirar carta de conducao");
}

if-else

O if nos permite executar um codigo se e somente se uma data condicao for satisfeita, caso contrario, outra porcao de codigo sera executada.

Exemplo:

let idade = 14;
if(idade >= 18) {
console.log("Ja pode tirar carta de conducao");
} else {
console.log(`Ainda nao tem idade suficiente para tirar carta de conducao. Aguarde mais ${18 - idade}`);
}

switch

O switch e' normalmente usado para escolher uma opcao no meio de varias opcoes disponiveis.

Uma aplicacao para o switch sao os menus.

Estrutura switch


switch(variavel) {
case 'opcao1':
//codigo que sera executado se a opcao escolhida por a 'opcao1';
break;
case 'opcao2':
//codigo que sera executado se a opcao escolhida por a 'opcao2';
break;
....
default:
//codigo que sera executada se a opcao escolhida nao for nenhuma das listadas acima
break;
}

Obs:

Usamos o break porque quando a opcao e' encontrada as opcoes seguintes nao devem ser testadas.
