JavaScript (Parte 6)

Métodos de Strings

length

length retorna o numero de caracteres de uma string.

Exemplo:

let texto = "Hello, world!";
console.log(texto.length);

Saida: 13

substring(inicio, fim)

substring extrai uma parte de uma string e retorna a parte extraida da string.
O inicio e o fim da parte extraída e' definida pelos parâmetros 'inicio' (inclusivo) e fim (exclusivo)

Exemplo:

let texto = "Hello, world!";
let novoTexto = texto.substring(0, 5);
console.log(novoTexto);

Saida: Hello

substr(inicio, tamanho)

substr e' similar ao substring a diferença e' que o segundo parâmetro indica o tamanho da parte extraída no string.
O inicio e o fim da parte extraída e' definida pelos parâmetros 'inicio' (inclusivo) e fim (exclusivo)

Exemplo:

let texto = "Hello, world!";
let novoTexto = texto.substr(7, 6);
console.log(novoTexto);

Saida: world!

replace(parteAntiga, parteNova)

replace substitui uma parte de uma string (parteAntiga) por uma nova (parte nova).

Notas:

  • replace nao substitui a string inicial, mas sim, gera uma nova string.
  • replace apenas substitui a primeira ocorrência
  • replace e' sensível a maiúsculas

Exemplo:

let texto = "Ola, João. Ola, Paulo.";
let novoTexto = texto.replace("Ola", "Oi");
console.log(`Nova String: ${novoTexto}`);
console.log(`Antiga String: ${texto}`);

Saída:
Nova String: Oi, João. Ola, Paulo.
Antiga String: Ola, João. Ola, Paulo.

Para substituir todas as ocorrências, use uma expressão regular com o flag /g.

Notas:

  • Expressões regulares sao escritas sem aspas.

Exemplo:

let texto = "Ola, João. Ola, Paulo.";
let novoTexto = texto.replace(/Ola/g, "Oi");
console.log(`Nova String: ${novoTexto}`);
console.log(`Antiga String: ${texto}`);

Saída:
Nova String: Oi, João. Oi, Paulo.
Antiga String: Ola, João. Ola, Paulo.

toUpperCase()

toUpperCase() converte toda String para maiúsculas.

Exemplo:

let texto = "Ola, Mundo!";
console.log(texto.toUpperCase());

Saída: OLA, MUNDO!

toLowerCase()

toLowerCase() converte toda String para minusculas.

Exemplo:

let texto = "Ola, Mundo!";
console.log(texto.toLowerCase());

Saída: ola, mundo!

trim()

trim() Elimina espaços no inicio e no fim de uma String.

Exemplo:

let texto = " Ola, Mundo! ";
console.log(texto.trim());

Saída: Ola, mundo!

trimStart()

trimStart() Elimina espaços e no inicio de uma String.

Exemplo:

let texto = " Ola, Mundo!";
console.log(texto.trimStart());

Saída: Ola, mundo!

trimEnd()

trimEnd() Elimina espaços e no fim de uma String.

Exemplo:

let texto = " Ola, Mundo!";
console.log(texto.trimEnd());

Saída: Ola, mundo!

charAt(indice)

charAt() retorna o caractere localizado no índice especificado.

Exemplo:

let texto = "Ola, Mundo!";
let letra = texto.charAt(5);
console.log(letra);

Saída: M

split(separador)

split() converte uma String para um array.
O parâmetro (separador) do método e' que divide os elementos do array na String

Exemplo:

let texto = "HTML, CSS, JavaScript";
let array = texto.split(',');
console.log(array);

Saída: [HTML, CSS, JavaScript]

Exemplo:

let texto = "Laranja Banana Papaia";
let array = texto.split(' ');
console.log(array[1]);

Saída: Banana
