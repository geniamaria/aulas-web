JavaScript (Parte 1)

Definição de Script e Introdução ao JavaScript

JavaScript: E' uma linguagem de programacao usada para manipular elementos na web.

Script: E' um pequeno programa que e' inserido dentro de outro programa

  • Console.log(): Serve para mostrar alguma informacao no consola do browser (E' geralmente usado para testes)
  • prompt(): E' usando para recolher alguma informacao atraves de uma caixa de dialogo
  • alert(): E' usado para exibir uma informacao em uma caixa de dialogo.

Variaveis e declaracao de variaveis

Regras para a delcaracao de variaveis

  1. os nomes das variaveis devem iniciar com uma letra, underscore (_) ou simbolo de dolar ($)
  2. os nomes das variaveis nao podem iniciar com um numero ou um outro caractere especial
  3. os nomes das variaves nao podem conter espacos
  4. o JavaScript e' sensivel a maiusculas (case sensitive), i.e, Variavel e' diferente de variavel.

Declaracao de variaveis

As variaveis podem ser declaradas usando const, var ou let

const

Usamos o const quando declaramos uma constante (o valor nao muda).

Exemplo: const pi = 3.14;

var

Usamos o var quando declaramos uma variavel (o valor pode ser mudado).

Exemplo: var area = 23.54;

let

Usamos o let quando declaramos uma variavel (o valor pode ser mudado).

Exemplo: let nome = "Ana";

Diferencas entre o var e o let

  • usando o let nao e' posssivel aceder uma variavel antes de inicializa-la (ocorre um erro). Mas com o var e' possivel e dara' o resultado indefinido
  • O let e' limitado pelos escopos do programa enquanto o var nao.
  • E' recomendado usar let

Tipos de dado

O JavaScript e' uma linguem de tipagem dinamica.

Number

Number: Tipo de dado numerico que representa interiros e pontos flutuates (float)

Exemplos:

let idade = 12; const pi = 3.14;

String

String: Representa uma sequencia de caracteres (textos)

Obs: Textos devem estar entre aspas duplas (""), aspas simples ('') ou apostrofre(``)

Escape(\): E' usada quando queremos imprimir caracteres especiais (aspas, nova linha, tab, etc)

Alguns caracteres especiais:

  • \n: Quebra de linha
  • \t: Tab
  • TPC: Pesquisar outros caracteres especiais.

Exemplos:

let nome = "Ana"; let estadoCivil = "Solteira";

Concatenacao (+): E' a juncao de duas ou mais strings ou uma string e um numeros

boolean

Boolean: E' usando para representar valores booleanos (verdadeiro ou false)

Exemplos:

let isMarried = true;
let isAlive = false;

undefined

Undefined: E' um tipo de dado que representa variaveis que foram declaradas mas nao inicializadas

Exemplos:

let nome;
console.log(nome); //undefined

null

null: Tal como o undefined, o null representa variaveis vazias.

typeof

typeof: Mostra do tipo de dado de um valor

Exemplos:

console.log(typeof "Joao"); //string
console.log(typeof 14); //number
console.log(typeof null); //object

Conversão e Coerção

Conversão: E' quando nos convertemos manualmente um tipo de dado para o outro.

  • Number(string): Converte uma string para um numero
    let variable = "2022";
    let number = Number(variable);
    console.log(variable, typeof variable, number, typeof number); //2022 string 2022 number
  • String(number): Converte um numero para uma string
    let numero = 10;
    let str = String(numero);
    console.log(numero, typeof numero, str, typeof str); //10 number 10 string
  • Boolean(valor): O que faz? Coloque um codigo como exemplo aqui. let variable = false; let bool = Boolean(variable); console.log(variable, typeof variable)

Coerção: E' quando o JS converte implicitamente (por baixo dos panos) os tipos de dado.

NOTAS:

  • Quando usamos adicao (+) o valor do tipo numerico e' convertido para o tipo string
  • Quando usamos subtracao (-) o valor do tipo string e' convertido para o tipo numerio
