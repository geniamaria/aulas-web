Eventos

Eventos sao accoes que ocorrem quando o usuario interage com a pagina web.

Sintaxe:

$(selector).evento(function() {
//o código que sera executado quando o elemento e' acionado.
});

Exemplo:

$("#div").click(function() {
$("#div").text('Botao clicado com sucesso!')
});

Resultado:

Clique-me

Eventos Gerados pelo documento

load

Este evento e' acionado quando a pagina e' carregada.

Exemplo:

$(window).on('load', function() {
$("#load-div").hide();
let name = prompt("Introduza o seu nome");
$("#load-div").text(`Ola, ${name}. Bem-vindo ao meu site`);
$("#load-div").fadeIn(3000);
});

Resultado: (No topo da pagina)

Eventos Gerados pelo mouse

click

Este evento e' acionado quando clicamos em um elemento

Exemplo:

$("#div").click(function() {
$("#div").text('Botao clicado com sucesso!')
});

Resultado:

Clique-me

MouseEnter

Este evento e; acionado quando colocamos o mouse por cima de um elemnto

Exemplo:

$("#enter-div").MouseOver(function() {
$("#enter-div").text('O mouse ja passou por cima deste elemento.')
});

Resultado:

O mouse ainda nao passou por cima desteelemento.

MouseLeave

Este evento e; acionado quando tiramos o mouse de cima de um elemnto

Exemplo:

$("#leave-div").mouseleave(function() {
$("#leave-div").text('O mouse ja saiu por cima deste elemento.')
});

Resultado:

O mouse ainda nao saiu por cima deste elemento.

Exemplo: MouseEnter e MouseLeave no mesmo elemento

$("#leave-div").mouseenter(function() {
$("#enter-leave-div").text('Mouse Enter.')
});

$("#leave-div").mouseleave(function() {
$("#enter-leave-div").text('Mouse Leave')
});

Resultado:

Passe o mouse em mim.

preventDefault()

O preventDefault() impede que eventos padrao ocorram.
No exemplo abaixo, ao clicar no link, o usuario nao sera redirecionado para outra pagina, inves disso um alert sera mostrado.

Exemplo:

$("#link").click(function (e) {
e.preventDefault();
alert('Este link me leva para: ' + e.target.href);
});
