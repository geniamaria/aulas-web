Eventos

Eventos sao accoes que ocorrem quando o usuario interage com a pagina web.

Sintaxe:

$(selector).evento(function() {
//o código que sera executado quando o elemento e' acionado.
});

Exemplo:

$("#div").click(function() {
$("#div").text('Botao clicado com sucesso!')
});

Resultado:

Clique-me

Eventos Gerados pelo documento

load

Este evento e' acionado quando a pagina e' carregada.

Exemplo:

$(window).on('load', function() {
$("#load-div").hide();
let name = prompt("Introduza o seu nome");
$("#load-div").text(`Ola, ${name}. Bem-vindo ao meu site`);
$("#load-div").fadeIn(3000);
});

Resultado: (No topo da pagina)

Eventos Gerados pelo mouse

click

Este evento e' acionado quando clicamos em um elemento

Exemplo:

$("#div").click(function() {
$("#div").text('Botao clicado com sucesso!')
});

Resultado:

Clique-me

dbclick

Este evento e' acionado quando clicamos duas vezes em um elemento

Exemplo:

$("#db-div").dbclick(function() {
$("#db-div").text('Voce clicou em mim duas vezes!')
});

Resultado:

Clique em mim duas vezes.

MouseOver

Este evento e; acionado quando colocamos o mouse por cima de um elemnto

Exemplo:

$("#hover-div").MouseOver(function() {
$("#hover-div").text('O mouse esta por cima deste elemento!')
});

Resultado:

O mouse nao esta por cima deste elemento!

MouseLeave

Este evento e; acionado quando tiramos o mouse de cima de um elemnto

Exemplo:

$("#hover-div").mouseleave(function() {
$("#hover-div").text('O mouse nao esta por cima deste elemento!')
});

Resultado:

O mouse esta por cima deste elemento!
