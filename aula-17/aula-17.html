Arrays

Array e' uma estrutura de dados onde podemos armazenar vários dados e posteriormente aceder esses dados.

Criacao de um array

Existem duas dormas de criar um array.

Metodo 1:

Sintaxe: let nomeVariavel = new Array(valor 1, valor 2, valor 3, ...);

Metodo 1:

Sintaxe: let nomeVariavel = [valor 1, valor 2, valor 3, ...];

Posicoes dos elementos de uma array

Os elementos de um array sao identificados por indices, comecando de zero (0). ou seja, o primeiro elemento tem indice 0, o segundo elemento tem indice 1 e por diante.

Exemplo:

let lista = ["um", "dois", "tres"];
let primeiroElemento = lista[0];
let segundoElemento = lista[1];
let terceroElemento = lista[2];
console.log(`Primeiro elemento: ${primeiroElemento}, Segundo elemento: ${segundoElemento}`);

Saida: Primeiro elemento: um, Segundo elemento: dois

Metodos para a manipulacao de arrays

length

O metodo length retorna o tamanho de um array.

Exemplo:

let lista = [1, 2, 3, 4]; console.log(lista.length); Saida: 4;

push

O metodo push adiciona um elemento no final de um array.

Exemplo:

let lista = [1, 2, 3, 4]; lista.push("cinco"); console.log(lista); Saida: [1, 2, 3, 4, "cinco"];

unshift

O metodo unshift adiciona um elemento no inicio de um array.

Exemplo:

let lista = [1, 2, 3, 4]; lista.unshift("zero"); console.log(lista); Saida: ["zero", 1, 2, 3, 4];
