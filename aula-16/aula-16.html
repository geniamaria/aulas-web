Estruturas de repeticao

Usamos as estruturas de repeticao quando queremos que uma terefa seja repetida um certo numero de vezes;

Estrurtura for

Sintaxe:
for(inicializacao; condicao; incremento/decremento) {
TODO: O codigo que sera executado multiplas vezes
}

Exemplo: Um codigo que ira imprimir a frase "Bom dia" 10 vezes

for(let i = 0; i < 10; i++) {
console.log("Bom dia");
}

Estrurtura while

Sintaxe:
inicializacao; while (condicao) {
TODO: o codigo que sera executado multiplas vezes.

incremento/decremento;
}

Exemplo: Um codigo que ira imprimir a frase "Bom dia" 10 vezes

let i = 0; while(i < 10) {
console.log("Bom dia");
i++;
}

Estrurtura do-while

Sintaxe:
inicializacao; do {
TODO: o codigo que sera executado multiplas vezes.

incremento/decremento;
} while (condicao);

Exemplo: Um codigo que ira imprimir a frase "Bom dia" 10 vezes

let i = 0; do {
console.log("Bom dia");
i++;
} while(i < 10);

Diferenca entre o while e do-while

A diferenca entre o while e do-while e' que o while so' executa se a inicializacao estiver dentro do intervalo da condicao, enquando que do-while sempre executa pelo menos uma vez (mesmo se a inicializacao nao estiver dentro do intervalo da condicao).

Funcoes

Funcoes sao blocos de codigo que se dedicam a execucao de uma certa tarefa.

Funcoes declaradas

Sintaxe:
function nomeFuncao(parametros : opcional) {
TODO: o codigo que a funcao executa

return valorRetornado (opcional)
}

Exemplo: Uma funcao que recebe dois numeros, calcula a soma dos dois numeros e retorna o resultado da soma.

function somarNumeros(nr1, nr2) {
let soma = nr1 + nr2;

return soma;
}

Para executar a funcao do exemplo acima, devemos chamar a funcao e passar para ela os dois numeros que desejamos somar. Isto e,

let resultadoSoma = somarNumeros(10, 15);
console.log(resultadoSoma);

Funcoes Anonimas

Estas funcoes nao possuem um nome.
Normalmente essas funcoes sao atribuidas a uma variavel para poder usa-las e devem e devem ter sempre um retorno.

Sintaxe:
function(parametros : opcional) {
TODO: o codigo que a funcao

return valorRetornado (obrigatorio)
}

Exemplo: Uma funcao que recebe dois numeros, calcula a soma dos dois numeros e retorna o resultado da soma.

let somarNumeros = function (nr1, nr2) {
let soma = nr1 + nr2;

return soma;
}

Para executar a funcao anonima do exemplo acima, devemos devemos atribuir a funcao a uma certa variavel e usar a variavel para passar os argumentos. Isto e,

console.log(somarNumeros(10, 15));

Arrow Functions

Estas funcoes nao possuem um nome e nem a palavra "function"
Normalmente essas funcoes sao atribuidas a uma variavel para poder usa-las e devem ter sempre um retorno.

Sintaxe:
(parametros : opcional) => {
TODO: o codigo que a funcao

return valorRetornado (obrigatorio)
}

Se a funcao tiver somente a linha de retorno, entao a sintaxe fica simplificada da seguinte forma

Sintaxe:
(parametros : opcional) => valorRetornado (origatorio)

Exemplo: Uma funcao que recebe dois numeros, calcula a soma dos dois numeros e retorna o resultado da soma.

let somarNumeros = (nr1, nr2) => {
let soma = nr1 + nr2;

return soma;
}

Para executar a funcao anonima do exemplo acima, devemos devemos atribuir a funcao a uma certa variavel e usar a variavel para passar os argumentos. Isto e,

console.log(somarNumeros(10, 15));
